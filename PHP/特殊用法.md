## 特殊用法



1. 動態的變數名稱
```
$foo = "bar" ;
$$foo = "baz" ;
echo $bar; // baz
```

2. 參考
```
$white = "snow";
$black =& $white;
$white = '123';
print $black; // 123
```

3. 巢狀宣告
```
function outer($a){
    function inner($b){
        echo $b ;
    }
    echo $a ;
}
outer('well');
inner('reader');
// 先呼叫外層函式才可呼叫內層函式
```
