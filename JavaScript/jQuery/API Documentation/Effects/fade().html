fade()改變物件的opacity(透明)/transparency(不透明)性質

$(selector).fadeIn(speed,callback):可选的 speed 参数规定效果的时长。它可以取以下值："slow"、"fast" 或毫秒。

$(selector).fadeOut(speed,callback):可选的 speed 参数规定效果的时长。它可以取以下值："slow"、"fast" 或毫秒。

$(selector).fadeTo(speed,opacity,callback):渐变为给定的不透明度（值介于 0 与 1 之间）

$(selector).fadeToggle(speed,callback):在 fadeIn() 与 fadeOut() 方法之间进行切换。
