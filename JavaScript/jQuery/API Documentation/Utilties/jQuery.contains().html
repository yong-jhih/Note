$.contains() 方法用于判断指定元素内是否包含另一个元素。即判断另一个DOM元素是否是指定DOM元素的后代。

$.contains( container, contained )

container:Element类型 指定可能包含其他元素的祖辈容器元素。

contained:Element类型 指定可能被其他元素包含的后代元素。

$("#check").click(function(){alert($.contains( document.documentElement, document.body ));})

$("#checkoutTR").click(function(){alert($.contains( document.body, document.documentElement ));})
